These questions and similar ones for all of Adrian's courses are very common questions asked by prospective students embarking on the their AWS certification journey. These questions inspired me to build this Webapp in the hopes that it helps prospective students with their planning and time management. For the curious ones, you can find the source code and architecture of the Webapp on GitHub
